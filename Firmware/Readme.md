## ⚙️ Firmware – Arduino Control

This firmware controls the SpectroDrive robot using an Arduino Mega.

### 🔧 Features

* Light tracking algorithm
* Motor control via L298N
* Bluetooth communication (HC-05)
* Sensor data processing (LDR, ultrasonic, tilt)

### 📡 Communication

* Bluetooth commands received via HC-05
* UART-based communication

### ▶️ Usage

1. Upload the code to Arduino Mega
2. Connect HC-05 module
3. Pair with mobile app
4. Send commands via Bluetooth
